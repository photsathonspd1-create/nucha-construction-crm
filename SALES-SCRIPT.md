# 📞 PART 1: SALES CALL SCRIPT

## ⏰ ขั้นตอนการโทร (ตาม SLA)

### 🔴 Lead Score ≥ 5 → โทรภายใน 5 นาที

---

### STEP 1: เปิดสาย (15 วินาทีแรก)

```
"สวัสดีครับ/ค่ะ คุณ [ชื่อลูกค้า] ใช่มั้ยครับ/คะ?

ผม/ดิฉัน [ชื่อเรา] จาก NUCHA INNOVATION ครับ/ค่ะ
โทรกลับเรื่องที่คุณ [ชื่อ] สนใจ [บริการที่เลือก]
ไม่ทราบว่าสะดวกคุยสักครู่มั้ยครับ/คะ?"
```

**หลักการ:**
- ทักชื่อ → สร้างความคุ้นเคย
- บอกชื่อบริษัท → ความน่าเชื่อถือ
- อ้างสิ่งที่เขาสนใจ → ไม่ใช่โทรขายของเย็นชา
- ถามความสะดวก → ให้เกียรติ

---

### STEP 2: Qualify (ถาม 4 คำถามหลัก)

```
1. "ไม่ทราบว่าตอนนี้มีโครงการแบบไหนอยู่ในใจครับ/คะ?"
   → ฟัง: ประเภทงาน (บ้าน/อาคาร/รีโนเวท)

2. "มีที่ดิน/สถานที่แล้วหรือยังครับ/คะ?"
   → ฟัง: พร้อมทำจริงแค่ไหน

3. "ตั้งงบประมาณไว้เท่าไหร่ครับ/คะ?"
   → ฟัง: งบจริง vs ความคาดหวัง

4. "อยากเริ่มเมื่อไหร่ครับ/คะ?"
   → ฟัง: ความเร่งด่วน
```

**หลักการ:**
- ถามเปิด → ไม่ใช่ถามปิด (yes/no)
- ฟังมากกว่าพูด (80/20)
- จดทุกอย่าง → ลง CRM ทันทีหลังวางสาย

---

### STEP 3: Present (นำเสนอ ตามบริการ)

#### ถ้า "รับเหมาก่อสร้าง"

```
"เข้าใจครับ/ค่ะ สร้างบ้าน/อาคาร
ที่ NUCHA เราดูแลครบทุกขั้นตอนเลยครับ/ค่ะ
ตั้งแต่ออกแบบ → ขออนุญาต → ก่อสร้าง → ส่งมอบ

ทีมเรามีสถาปนิกและวิศวกรประจำ
ไม่ได้ส่งช่างรับเหมาต่อครับ/ค่ะ

ที่สำคัญคือเราไม่บวกราคา
ส่งมอบตรงเวลา พร้อมรับประกันโครงสร้าง

ไม่ทราบว่าเคยคุยกับที่อื่นไว้บ้างมั้ยครับ/คะ?"
```

#### ถ้า "บิ้วอิน"

```
"บิ้วอินครัว/ตู้เสื้อผ้าใช่มั้ยครับ/คะ
ที่ NUCHA เราใช้วัสดุเกรด A ทั้งหมด
ไม้ MDF หุ้มลามิเนต ไม่ใช่ไม้ปาร์ติเกิล

ทีมช่างเราเป็นช่างประจำ ไม่ใช่ outsource
งานเรียบร้อย รับประกัน 3 ปีครับ/ค่ะ

ส่งรูปห้องมาให้ดูได้มั้ยครับ/คะ?
จะทำ 3D ให้ดูก่อน ฟรี ไม่มีค่าใช้จ่าย"
```

#### ถ้า "ออกแบบ"

```
"ออกแบบบ้าน/ภายในใช่มั้ยครับ/ค่ะ
ที่ NUCHA เรามีทีมดีไซเนอร์เฉพาะทาง
ทำ 3D Rendering ให้ดูก่อนสร้างจริง

ลูกค้าจะเห็นบ้านก่อนสร้างเลยครับ/ค่ะ
แก้ไขได้ไม่จำกัดรอบ จนกว่าจะพอใจ

ส่งแบบที่ชอบมาให้ดูได้มั้ยครับ/คะ?
จะประเมินราคาเบื้องต้นให้เลย"
```

#### ถ้า "ตกแต่ง / รีโนเวท"

```
"รีโนเวทบ้าน/คอนโดใช่มั้ยครับ/ค่ะ
ที่ NUCHA เราดูแลตั้งแต่รื้อจนแต่งเสร็จ
ไม่ต้องหาหลายเจ้าครับ/ค่ะ

เราจะเข้าไปดูหน้างานก่อน
ประเมินหน้างานจริง ไม่เดาราคา

ส่งรูปปัจจุบันมาให้ดูได้มั้ยครับ/คะ?
จะประเมินเบื้องต้นให้เลย"
```

#### ถ้า "บริหารโครงการ"

```
"บริหารโครงการจัดสรรใช่มั้ยครับ/ค่ะ
ที่ NUCHA เรามีทีม Project Manager ดูแล
ควบคุมงบ ควบคุมคุณภาพ รายงานทุกสัปดาห์

ไม่ต้องมานั่งคุมเองครับ/ค่ะ
เราดูแลให้ตั้งแต่ต้นจนจบ

นัดเข้าไปดูไซต์ได้มั้ยครับ/ค่ะ?
จะวางแผนงานให้ดูเบื้องต้น"
```

---

### STEP 4: Close (นัดหมาย)

```
"ถ้าสะดวก ผม/ดิฉันขอนัดเข้าไปดูหน้างาน
หรือคุยผ่านวิดีโอคอลก็ได้ครับ/ค่ะ

วัน [เสนอ 2 วัน] เวลา [เสนอ 2 เวลา]
สะดวกมั้ยครับ/คะ?"
```

**หลักการ:**
- เสนอ 2 ตัวเลือก → ไม่ใช่ถาม "ว่างเมื่อไหร่"
- นัด onsite ดีกว่า online (ได้เห็นหน้างานจริง)
- ถ้าไม่สะดวก → นัด LINE call

---

### STEP 5: ปิดสาย

```
"ขอบคุณมากครับ/ค่ะ คุณ [ชื่อ]
เดี๋ยวผม/ดิฉันส่งรายละเอียดทาง LINE ให้ดูนะครับ/คะ
แล้วเจอกันวัน [วันที่นัด] ครับ/ค่ะ

มีอะไรเพิ่มเติม LINE มาได้เลยนะครับ/คะ
ขอบคุณครับ/ค่ะ 🙏"
```

### ⚠️ OBJECTION HANDLING

#### "แพงไป"

```
"เข้าใจครับ/ค่ะ เรื่องงบประมาณเป็นสิ่งสำคัญ
ที่ NUCHA เราไม่ได้ถูกที่สุด แต่คุ้มค่าที่สุดครับ/ค่ะ

วัสดุที่ใช้เกรด A ทั้งหมด
ทีมช่างประจำ ไม่ใช่ outsource
รับประกันโครงสร้างตลอดอายุใช้งาน

ถ้าเทียบกับเจ้าที่ถูกกว่า
สุดท้ายต้องมาซ่อม แพงกว่าหลายเท่าเลยครับ/ค่ะ

ไม่ทราบว่างบประมาณที่ตั้งไว้เท่าไหร่ครับ/คะ?
จะปรับแบบให้เหมาะกับงบให้"
```

#### "ยังไม่พร้อม / แค่ดูไว้ก่อน"

```
"ไม่เป็นไรเลยครับ/ค่ะ ดูไว้ก่อนก็ดี
การสร้างบ้านต้องวางแผนนานครับ/ค่ะ

ส่งแบบบ้าน/ผลงานเราไว้ดูเล่นก่อนนะครับ/คะ
ถ้าพร้อมเมื่อไหร่ โทรมาได้เลย

ส่วนราคา... ถ้าปีนี้ตัดสินใจ
เราจะล็อคราคาวัสดุให้ครับ/ค่ะ
ปีหน้าวัสดุขึ้นราคาแน่นอน"
```

#### "มีเจ้าอื่นอยู่แล้ว"

```
"ดีเลยครับ/ค่ะ ถ้ามีเจ้าอื่นอยู่แล้ว
แนะนำให้เปรียบเทียบ 3 อย่างนี้ครับ/ค่ะ:

1. สัญญา — มีระบุวัสดุยี่ห้อไหนชัดเจนมั้ย?
2. ทีม — เป็นช่างประจำหรือ outsource?
3. รับประกัน — กี่ปี? ครอบคลุมอะไรบ้าง?

ถ้าอยากให้ช่วยดูสัญญาเจ้าอื่นให้
ส่งมาให้ดูได้เลยครับ/ค่ะ ไม่คิดเงิน"
```

#### "ต้องถามแฟน/ครอบครัวก่อน"

```
"เข้าใจครับ/ค่ะ เรื่องบ้านต้องตัดสินใจร่วมกัน
ส่งรายละเอียดให้แฟนดูก่อนนะครับ/คะ

แล้วนัดคุยกัน 3 คนพร้อมกันก็ได้ครับ/ค่ะ
จะได้อธิบายให้ทุกคนเข้าใจตรงกัน

วัน [เสนอวัน] สะดวกมั้ยครับ/คะ?"
```

#### "ไม่รับสาย / ไม่ตอบ"

```
→ Follow-up ทาง LINE (ดู PART 2)
→ โทรอีกครั้งวันถัดไป เวลาต่างกัน
→ ถ้า 3 วันไม่ตอบ → ส่งข้อความสุดท้าย
→ ถ้า 7 วันไม่ตอบ → mark "ไม่ตอบกลับ"
```

# 💬 PART 2: LINE FOLLOW-UP TEMPLATES

## Template 1: หลังวางสาย (ทันที)

```

ผม/ดิฉัน {ชื่อเรา} จาก NUCHA INNOVATION ครับ/ค่ะ

📋 สรุปเบื้องต้น:
• บริการ: {บริการ}
• งบประมาณ: {งบ}

📎 ผลงานของเรา: {ลิงก์}

มีอะไรเพิ่มเติม LINE มาได้เลยนะครับ/คะ 😊
```

## Template 2: ส่งผลงาน/Portfolio (วันถัดไป)

```

```

เจอกันพรุ่งนี้ครับ/ค่ะ 😊
```

```

## Template 5: Follow-up 7 วัน (ไม่ตอบ)

# 🧠 PART 3: OBJECTION HANDLING CHEAT SHEET

| ข้ออ้าง | ตอบกลับ |
|---------|---------|
| แพงไป | เปรียบเทียบคุณภาพ + รับประกัน + ค่าซ่อมระยะยาว |
| ยังไม่พร้อม | ส่งผลงาน + ล็อคราคาวัสดุ |
| มีเจ้าอื่น | เปรียบเทียบ 3 อย่าง (สัญญา/ทีม/รับประกัน) |
| ต้องถามแฟน | นัดคุย 3 คนพร้อมกัน |
| ไม่รับสาย | LINE + โทรเวลาต่างกัน + 3 วัน mark |
| ขอดูผลงานก่อน | ส่ง portfolio ทันที |
| อยากได้ถูกกว่า | ปรับแบบให้เหมาะกับงบ |
| กลัวผิดหวัง | รับประกัน + ผลงานจริง + รีวิวลูกค้า |
